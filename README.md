# Smartphone-based-Audiometry
Audiometry (Hearing Test) using Smartphone (iOS 10+)
 
## Overview

This GitHub repository provides codes for Hearing Test Application written in swift for iOS.
Application performs hearing test for left and right ear and generate an Audiogram for a user. The hearing threshold for each octave frequency will be tsted for both ears. Calibration 
for Apple earphone's have been done and are perfect for the use with the abpve application.

** Acuracy of the audiogram has been tested with normal hearing persons and the audiogram results are accurate within +/- 10 dB considering human take retake errors. We strongly suggests
that if you have high degree of hearing loss than please consult professional Audiologist.

## Users Guides
A User's Guide is provided describing the implementation in Xode. 

## Requirements 
iOS v 10+, 
Apple Earphones for listening, 
A Quite room for testing and, 5 minutes of patience!
